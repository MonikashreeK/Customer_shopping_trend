# Customer_shopping_trend
Data analysis of Customer and their shopping trend using Pyhton, SQL and Power BI.

Data Analytics Project

Overview
This project demonstrates an end-to-end data analytics workflow, from data loading and cleaning to analysis, visualization, and reporting. The goal is to extract meaningful insights from raw data using Python, SQL, and business intelligence tools, and present the findings in a clear and professional manner.

Dataset
* Source: From Kaggle
* Format: CSV / Excel / Database table
* Description: The dataset contains structured data used to analyze trends, patterns, and key metrics relevant to the business problem.

Tools & Technologies
* Python (Pandas, NumPy, Matplotlib, Seaborn)
* SQL (PostgreSQL / MySQL / SQL Server)
* Power BI (Dashboard creation)
* Microsoft Excel (Optional preprocessing)
* PowerPoint (Final presentation)
* Git & GitHub (Version control)

Project Steps
1. Data Loading
   * Imported dataset into Python for analysis.
   * Loaded cleaned data into a SQL database.

2. Exploratory Data Analysis (EDA)
   * Analyzed data distributions, trends, and correlations.
   * Identified missing values and outliers.

3. Data Cleaning
   * Handled missing and inconsistent data.
   * Standardized formats and corrected data types.

4. SQL Analysis
   * Wrote SQL queries to extract insights and perform aggregations.
   * Joined multiple tables where required.

5. Dashboard Development
   * Built an interactive Power BI dashboard.
   * Visualized KPIs, trends, and comparisons.

6. Reporting & Presentation
   * Created a structured report summarizing findings.
   * Designed a PowerPoint presentation for stakeholders.

Dashboard
The Power BI dashboard includes:
* Key performance indicators (KPIs)
* Trend analysis
* Category and segment comparisons
* Interactive filters for deeper exploration

Results & Insights
* Identified key trends and patterns in the data
* Highlighted performance drivers and problem areas
* Provided actionable insights to support decision-making
